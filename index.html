Compare Instagram Followers

How it works

  1. Select Download or transfer information.

  2. Select a single Instagram account.

  3. Select Some of your information.

  4. Scroll to Followers and following and select it.

  5. Choose Download to device.

  6. Set Date range to All time.

    Set Format to JSON.

    Select Create files.

  7. Wait for Instagram to generate the files.

    This may take a few minutes. I don’t know why.

  8. Select Download, then enter your password.

  • Your data stays on your device. We never upload or store your information on any servers.
  • All processing happens right here in your browser.
  • You can use this tool offline once the page has loaded.
  • We only read the followers and following lists from your Instagram data. Nothing else is accessed.

Still not sure? The source code is available and accepting contributions!
